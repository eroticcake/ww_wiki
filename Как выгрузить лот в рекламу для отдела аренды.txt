Как выгрузить лот в Циан
1. Зайдите в карточку лота.
2. Нажмите кнопку «Выгрузка на площадки».
3. Поставьте галочку «Опубликовано». Если хотите снять лот с рекламы, уберите галочку.
4. В окне ниже выбираем «Whitewill Аренда».
5. Заголовок доступен только для «Топ3» размещения. Писать заголовок Caps lock нельзя.
6. Заполните адрес в формате: Россия, Москва, Казарменный переулок,3
7. Выберите тип размещения для лота: «Платное или «Топ3». Ставку аукциона можно применить только к «Топ3». Но учитывайте, что кол-во «Топ3» ограничено, применять его нужно только к ликвидным объектам.
8. Выберите кол-во комнат.
9. Выберите тип комнат.
10. Укажите площадь комнат через пробел, от большей к меньшей: 23 20 11
Как правильно указать площадь —   https://wiki.yandex.ru/homepage/cian/obshhajaizhilajaploshhad/
11. Укажите площадь кухни.

12. Укажите жилую площадь.
13. Выберите тип здания.
14. Выберите тип парковки.
15. Выберите вид из окон.
16. Бонус для контрагента — если мы хотим указать, что делимся бонусом с брокером, например, готовы отдать ему 50% от нашей комиссии.
17. Выберите кол-во пассажирских и грузовых лифтов.
18. Выберите кол-во раздельных и совмещенных санузлов.

19. Комиссия для клиента — указываем только в том случае, когда у нас есть внешняя комиссия, которую нам должен заплатить клиент. В этом случае, мы указываем комиссию и для агента. Циан сделал это для того, чтобы если агент привел нам клиента, тоже понимал, что комиссия для клиента не отменяется и его клиент все равно ее платит.
20. Укажите комиссию для контрагента.
21. Выбираем выгрузку от себя, если хотим, чтобы клиенты видели ваше ФИО и фотографию.

22. Использовать общий телефон — не ставим галочку.
Описание и фотографии комплекса добавляем по желанию. Всегда проверяем описание жк на наличие информации о других квартирах. Если текст будет как на картинке ниже, циан отправит объявление в ошибку.

23. Дополнительные поля заполняем по желанию.

24. Чтобы изменить порядок — перетащите фотографию в нужное место.
Чтобы скрыть — нажмите на фотографию, она станет серой и публиковаться не будет.

25. Нажмите «Сохранить».
Как выгрузить лот в Авито
1. Поставьте галочку «Опубликовано».
2. В окне ниже выбираем «Основной аккаунт».
3. Выберите кол-во комнат.
4. Укажите тип ремонта.
5. Укажите возможность проживания с детьми/животными.
6. Выберите кол-во пассажирских и грузовых лифтов.
7. Укажите комиссию для клиента

8. Укажите комиссию для контрагента
9. Выберите тип комнат.
10. Выберите тип парковки.
11. Выберите тип здания.
12. Площадь кухни подтянется автоматически, если уже указали ее во вкладке «Циан»
13. Жилая площадь подтянется автоматически, если уже указали ее во вкладке «Циан»
14. Использовать общий телефон — не ставим галочку.
Описание и фотографии комплекса добавляем по желанию. Всегда проверяем описание жк на наличие информации о других квартирах.
15. Дополнительные поля заполняем по желанию.

16. Нажмите «Сохранить».
Как выгрузить лот в Яндекс
1. Поставьте галочку «Опубликовано».
2. В окне ниже выбираем «Аренда».
3. Адрес подтягивается из карточки объекта.
4. Выберите кол-во комнат.
5. Укажите бонус для контрагента.
6. Укажите комиссию для клиента.
7. Укажите комиссию для контрагента.
8. Выберите вид из окон.
9. Выберите тип здания.

10. Отмечаем галочкой следующие параметры, если они подходят по характеристикам к лоту: Свободная планировка, Коммунальные услуги включены, интернет, холодильник, мебель на кухне, кондиционер, мебель, лифт, мусоропровод, консьерж, закрытая территория.
11. Использовать общий телефон — не ставим галочку.
Описание и фотографии комплекса добавляем по желанию. Всегда проверяем описание жк на наличие информации о других квартирах.
12. Дополнительные поля заполняем по желанию.

13. Нажмите «Сохранить».
